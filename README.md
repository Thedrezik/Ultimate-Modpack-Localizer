# 🌍 MineAI Translator (EN -> RU)

Мощный и умный скрипт на Python для автоматического перевода модов и целых сборок (модпаков) Minecraft **с английского на русский язык**. Создано специально для ру-комьюнити!

В отличие от простых парсеров, этот инструмент не ломает игровой код, умеет отличать технические символы от текста и поддерживает перевод не только через **Google Translate**, но и через **Локальные Нейросети (LLM)** для получения высококачественного и литературного перевода лора.

## ✨ Главные особенности
* **Переводит всё:** Интерфейс модов, предметы, внутриигровые справочники (Patchouli, Lexicon) и квесты (FTB Quests).
* **Безопасность кода:** Маскирует цветовые коды (`§a`), переменные (`%s`) и макросы книг перед отправкой в ИИ, возвращая их целыми.
* **Аудит сборки:** Умеет сканировать папку с игрой и показывать в процентах, сколько строк на английском уже переведено на русский для каждого мода.
* **Умный пропуск:** Автоматически пропускает моды, которые уже переведены на 90% и более.
* **Пакетный перевод:** Отправляет текст пачками по 20 строк, ускоряя процесс перевода в несколько раз.
* **Встроенный ИИ сервер:** Автоматически скачивает `koboldcpp` и поднимает локальный сервер для нейросети — вам нужно только закинуть модель.

## 🚀 Установка и запуск

1. Убедитесь, что у вас установлен **[Python 3.10+](https://www.python.org/)** (при установке обязательно поставьте галочку "Add Python to PATH").
2. Скачайте этот репозиторий (кнопка `Code -> Download ZIP`) и распакуйте.
3. Поместите файлы `translator.py`, `start.bat` и `requirements.txt` **прямо в главную папку вашего профиля Minecraft** (туда, где лежат папки `mods` и `config`).
4. Запустите `start.bat`. Он сам установит нужные библиотеки и откроет меню.

## 🧠 Как переводить через Нейросеть (ИИ)
Перевод через Google работает сразу из коробки. Если вы хотите идеальный литературный русский язык без "машинного" акцента:

1. Создайте папку `AI` рядом со скриптом.
2. Скачайте любую подходящую LLM в формате `.gguf` (например, с HuggingFace). 
   * *Рекомендуется: `Qwen2.5-14B-Instruct-Q4_K_M.gguf` (требует 16 ГБ ОЗУ) или `Qwen2.5-7B-Instruct...` (требует 8 ГБ ОЗУ).*
3. Положите скачанный `.gguf` файл в папку `AI`.
4. Запустите скрипт, выберите нужный режим и укажите "Локальная Нейросеть" в качестве движка. Программа сама подхватит модель и начнет работу!

## 🛠️ Как это работает?
Скрипт читает `.jar` архивы на лету. Он берет оригинальный английский файл `en_us.json`, очищает его от мусорных комментариев, вытаскивает строки, переводит их на русский и аккуратно упаковывает обратно как `ru_ru.json`. Для квестов (`.snbt`) создаются резервные копии `.bak`.
